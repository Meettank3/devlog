# DevLog 🚀
A daily standup tracker for developers with AI-powered weekly summaries.

🔗 Live: https://devlog-lemon.vercel.app

## Features
- 🔐 Auth (Login, Signup, Forgot Password)
- 📝 Daily standup logging
- 🤖 AI weekly summary (Groq LLaMA)
- 🔥 Streak tracking
- 📱 Mobile responsive

## Tech Stack
- Next.js
- Tailwind CSS
- Supabase
- Groq AI

## Getting Started
1. Clone the repo
2. Create `.env.local` with your keys:
NEXT_PUBLIC_SUPABASE_URL=your_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key
GROQ_API_KEY=your_key
3. Run `npm install`
4. Run `npm run dev`
