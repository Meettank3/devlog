# DevLog 🚀
A daily standup tracker for developers.

## Features
- 🔐 Auth (Login, Signup, Forgot Password)
- 📝 Daily standup logging (coming soon)
- 🤖 AI weekly summary (coming soon)
- 📊 GitHub commit sync (coming soon)

## Tech Stack
- Next.js
- Tailwind CSS
- Supabase

## Getting Started
1. Clone the repo
2. Create `.env.local` with your Supabase keys
3. Run `npm install`
4. Run `npm run dev`
